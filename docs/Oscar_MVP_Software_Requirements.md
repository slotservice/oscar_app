OSCAR MVP Software Requirements
Product Overview
Oscar is a tablet‑friendly wastewater plant operations app that guides operators through a structured daily walk‑through checklist, captures lab and process data, records observations, and provides simple rule‑based operational suggestions. The first version replaces paper rounds and improves operational consistency.
Core Workflow
Login → Select Plant → Start Daily Round → Complete Checklist Sections → Enter Lab / Process Data → View Suggestions → Enter Issues / Actions → Sign Off → Save Daily Record
Core Sections
-Arrival / Grounds
-Mechanical Equipment
-Process Observations
-Housekeeping
-Labs & Operating Data
-Suggestions
-Issues / Follow‑Up
-Summary / Sign Off
Key Features
-30–35 daily checklist items
-Status buttons (OK / Attention / N/A)
-Required notes on abnormal items
-Grouped lab data entry
-Selectable observation tags
-Simple rule‑based suggestion engine
-Daily history storage
-Configurable plant thresholds
-Auto‑save
-Tablet‑friendly interface

Example Suggestion Logic
Low DO → Check aeration performance.
Elevated Ammonia → Review biomass condition and loading.
Cloudy Clarifier → Review settling and RAS operation.
Low MLSS → Biomass inventory may be low.
Temperature Drop → Monitor biological performance.
